## 3. Semantic Themes

Used in the References block of each direction. Second position in the reference card: `◉ Direct analogue · Insight · connection tag`.

| Theme | What it covers |
|---|---|
| Insight | Moment of understanding, revelation, decoding, extracting meaning from noise |
| Voice | Sound, speech, recording, audio, voice as material |
| Trust | Reliability, authority, institutional confidence, C-level gravitas |
| Data | Metrics, visualisation, patterns, generative / algorithmic identity |
| Growth | Progress, development, trajectory, transformation |
| Craft | Tactility, materiality, faceting, craftsmanship |
| Warmth | Humanity, approachability, relational tone |
| System | Modularity, grid, responsive, scalable architecture |

Pick the minimum necessary set per direction (typically 3–4 themes). Ideally each direction spans at least 3 different themes.

## 4. Diversity Quotas

Hard limits verified before shipping the report. If unmet — go for another search round (max 3 rounds), then honestly mark ⚠️.

| Criterion | Limit | Check |
|---|---|---|
| Geographic | ≥ 25% of brands not from US/UK | By country of brand origin |
| Temporal | ≥ 20% of cases older than 3 years from current date | By rebrand / launch year |
| Studio | ≤ 2 appearances per studio across the whole report | Running `[STUDIOS]` counter |
| Industry per direction | ≥ 3 different industries in each direction's References | By brand category |
| Registry | ≤ 2 appearances of the same brand across the whole report | Via `[REGISTRY]` marker |

## 6. Perceptual Map Axes

Validated axis pairs by category. Pick the one that actually separates brands — not the one that sounds nice. If the category doesn't match any row — formulate your own pair and justify it in prose.

| Category | Recommended axis pair |
|---|---|
| Enterprise SaaS | Warm ↔ Cold × Accessible ↔ Authoritative |
| Pricing / Revenue tools | Corporate ↔ Expressive × Human ↔ Technocratic |
| HR-tech / People analytics | Operational ↔ Aspirational × Quiet ↔ Loud |
| Consumer tech | Minimal ↔ Maximal × Familiar ↔ Strange |
| Craft / Material brands | System ↔ Craft × Silent ↔ Loud |
| AI-native products | Tool ↔ Agent × Technical ↔ Narrative |
| Hospitality / Members clubs | Exclusive ↔ Democratic × Traditional ↔ Forward |
| Finance / Fintech | Institutional ↔ Challenger × Rational ↔ Emotional |

## 7. Often Blocked / Unreliable Domains

Do not use as primary link targets. If a blocked domain appears in a useful search snippet — quote the snippet and include the URL, but note it may not open.

| Domain | Problem |
|---|---|
| `underconsideration.com/brandnew` | Frequently egress-blocked; returns EGRESS_BLOCKED |
| `the-brandidentity.com` | Frequently egress-blocked |
| `brandfetch.com` | Redirects; doesn't serve direct logo pages to agents |
| `logotyp.us` | Most constructed URLs 404; pattern `/logo/{brand}` rarely works |
| `seeklogo.com` | Sometimes returns bot-challenge pages |

If a blocked domain appears in search results with useful snippet info — quote the snippet and use `itsnicethat.com` or `bpando.org` as the primary link instead.

## 8. For Notion Output

When writing links to Notion pages:
- Never use the raw domain as link text (Notion truncates long URLs used as text)
- Format: `[Studio Name – Project](URL)` or `[Brand Name rebrand coverage](URL)`
- Test that the URL is under 300 characters
- Inline code blocks cannot contain hyperlinks — put links in separate properties or rich text blocks
